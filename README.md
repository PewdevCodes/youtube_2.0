#                                                               VIDEXA

## Project Overview

VIDEXA is a YouTube clone built with React and Vite, providing a modern video streaming platform with a clean, responsive interface.

## Features

- Fast video streaming and playback
- Search and discovery functionality
- User-friendly video recommendations
- Responsive design for mobile and desktop
- Real-time updates with HMR during development

- Node.js (v16+)

```bash
npm install
```

### Development

